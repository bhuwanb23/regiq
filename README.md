## 🚀 **Getting Started**

### 📋 **Prerequisites**

```bash
📱 Node.js 18+ and npm/yarn
🐍 Python 3.9+
🐳 Docker & Docker Compose
📊 PostgreSQL 14+
🔄 Redis 6+
```

### ⚡ **Quick Setup**

#### 🎯 **1. Clone the Repository**
```bash
git clone https://github.com/your-username/regiq.git
cd regiq
```

#### 🐳 **2. Docker Setup (Recommended)**
```bash
# Start all services
docker-compose up -d

# View logs
docker-compose logs -f
```

#### 📱 **3. Mobile App Setup**
```bash
cd regiq
npm install
npx expo start
```

#### ⚡ **4. Backend Setup**
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

#### 🤖 **5. AI Services Setup**
```bash
cd ai-ml
pip install -r requirements.txt
python -m services.gemini.main
```
